Plugin for automated data migration from Umbraco to WordPress. Convert Umbraco to WordPress easily and fast.

== Description ==

CMS2CMS: Umbraco to WordPress Migration Plugin has been specifically developed with an end-user in mind to make your conversion process across these two platforms as effortless and precise as possible. This handy online service expects no programming experience to work with.

= HOW THIS PLUGIN HELPS: =
- Intuitiveness – CMS2CMS requires no special skills to operate it so that even newbies and first-time users feel comfortable to how it works;
- Speed – due to the automated process, migration across these two software packages is a matter of a few minutes (up from 15 min depending on a website size);
- Free Demo Available – CMS2CMS provides its users with a possibility to see the website migration process in action and to estimate the results;
- Complete Migration price depends on how many content items you want to move to your New WordPress website.


= WHAT WILL BE MOVED FROM UMBRACO TO WORDPRESS? =
* Pages along with the content included there;
* Attachments (images and videos);
* Images included in the content body.

= PRESERVED SEO JUICE AFTER THE MIGRATION =
* Migration of SEO friendly URLs;
* Migration of keywords, metadata/meta description;
* Automated 301 redirects from existing Umbraco site to the new WordPress website.


= COMPATIBLE WITH: WORDPRESS 4.1.X – 5.0.X. =
- Our support team is available live 24/7.
- For more customized migration type, CMS2CMS offers data migration service packages to order.

== Installation ==
1. Download the plugin zip file.
2. Go to the Admin Panel -> Plugins -> Add New -> Upload Plugin.
3. Choose the downloaded file in zip format and click the “Install Now” button.
4. Go to Admin -> Plugins, activate the “CMS2CMS: Automated Umbraco To WordPress Migration Plugin”.
5. Register an account or sign in to the existing one.
6. Provide your Existing website URL address.
7. Click ‘Verify Connection’. You’ll be redirected to the CMS2CMS website to proceed with your migration.
